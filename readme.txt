Все необходимые для компиляции файлы приводятся здесь.
train.csv -  обучающая выборка,
private_test.csv - приватный тест,
_FF2.csv - последний сабмит, приводится для проверки воспроизводимости программы в конце,
sample_submission пример сабмита с публичного теста. применял до финала соревнования,
russian.txt - большой набор с русскими словами во множество форм из интернета (используется для корректировки словаря, не самой модели! код для загрузки уже указан в модели),
Итоговая_модель.ipynb - программа с итоговой моделью записанной в функциональном виде (не ООП). 

Для получения того же результата в теории должно быть достаточно загрузить код и все приложенные файлы в google colab и поменять адреса файлов т.к. я выгружал их из google drive (так быстрее).
В конце программы есть несколько блоков для проверки повторяемости. Если у вас по каким-то причинам они отличаются или модель не работает прошу написать мне в telegram @PavelGladkih
